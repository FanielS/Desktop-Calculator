# Desktop-Calculator

## This is a simple calculator that performs all four basic arthimetic operations in math.
- Addition
- Subtraction
- Multiplication
- Division

## Features
- Perform operations
- Hanlde illegal operations
- Reset and Delete functions
- Format result etc. . .
